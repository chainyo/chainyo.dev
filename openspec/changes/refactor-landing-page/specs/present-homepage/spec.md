## ADDED Requirements
### Requirement: Founder-led AI engineering positioning
The homepage SHALL present the site owner as a founder-led AI engineer focused on production-grade B2B systems.

#### Scenario: Visitor lands on homepage
- **WHEN** a visitor views the hero content
- **THEN** the messaging states founder-led AI engineering and B2B focus

### Requirement: Current focus context
The homepage SHALL describe the current primary client engagement as the main focus without implying generic freelance availability.

#### Scenario: Visitor reads the current focus section
- **WHEN** the visitor reaches the current focus section
- **THEN** the copy highlights a primary B2B client focus and selective commitments

### Requirement: Stack tools highlight
The homepage SHALL display a stack tools block with linked logos for key tools used.

#### Scenario: Visitor scans the hero area
- **WHEN** the visitor views the hero section
- **THEN** a stack tools block lists at least two tools with links

### Requirement: Projects as proof of execution
The homepage SHALL showcase independent projects as secondary to the main B2B work.

#### Scenario: Visitor scans the projects section
- **WHEN** the visitor reaches the projects section
- **THEN** the page lists at least one project with a short description and link

### Requirement: Reach-out channel
The homepage SHALL provide a clear call to action for reaching out.

#### Scenario: Visitor wants to contact
- **WHEN** the visitor looks for contact options
- **THEN** a CTA provides at least one contact channel (email or social)

### Requirement: Updated metadata
The homepage SHALL expose metadata aligned with the new positioning.

#### Scenario: Page metadata is read
- **WHEN** the page is loaded or shared
- **THEN** the meta title and description reflect the founder-led B2B AI engineering narrative
